# Automation Test Dashboard

## 🚀 Features
- Run automated test cases (API/UI)
- Dashboard with Pie & Bar charts
- Store results in MySQL
- Screenshot on failure
- Create test cases from UI

## 🛠 Tech Stack
- Java + Spring Boot
- MySQL
- Chart.js
- REST APIs

## ▶️ Run Project
1. Clone repo
2. Run Spring Boot
3. Open http://localhost:8080

## 📊 Dashboard
- Total / Pass / Fail
- Pie chart
- Execution time bar chart
